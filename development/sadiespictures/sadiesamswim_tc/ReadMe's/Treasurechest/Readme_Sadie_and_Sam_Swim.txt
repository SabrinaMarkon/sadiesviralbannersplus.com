Sadie and Sam Swim for Sadie and Sam. Bikini top and bottom, trunks, swim shorts, cap, mask, snorkel, flippers and swim ring. Three sets of textures for all and an extra one for the bikini.

To fit the waist towel over Sadie's bikini bottom, use the morph "fit to Sadie" then scale to 109.

For use in Poser 7 and possibly Poser 6. Not tested in DAZ Studio.

--------------------------------------

You will need:-  "Sadie" by 3D Universe
		 "Sam"   by 3D Universe

--------------------------------------

Installation Instructions:

Install using an unzip program directly into your Poser 7 library.
